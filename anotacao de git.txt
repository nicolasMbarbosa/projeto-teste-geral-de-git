Códigos de git
 git version - mostra a versao

git --help - verificar informações do git

git status - relatorio de controle de versao

git add "nome completo do arquivo"

atenção aos nomes em vermelho

como adicionar varios docs de uma vez?
 git add . - salva e adiciona novos arquivos

git commit -m "commit inicial" - comando que permite realizar commits de salvamento de versão, na primeira vez solicita a concatenação com a conta de email Git
 com os comandos
git config --global user.email "email@email.com"
git config --global user.name "Your Name"

modificação de código e salvando novos branches

git "push" que faz subir para o repositorio de GIT tudo aquilo que você tem de versões salvas ou que ainda não foi salvo.

 git reflog é o comando git que tras o historico de versoes ja salvas

Como navegar entre versões?
pode se usar o git reflog para ver quantas versões o código tem.

e usar o "git reset --hard (código da versão que é apresentado no reflog)"
